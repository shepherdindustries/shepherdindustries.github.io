README for USPS Zone Map Tool

Description
The USPS Zone Map Tool is a web-based utility designed to provide users with a visual and informative representation of the USPS shipping zones based on the 'from' and 'to' ZIP codes provided. The tool provides an interactive map, allowing users to view the various USPS shipping zones.

Features/Benefits
- User-Friendly Interface: Simple input fields to enter 'from' and 'to' ZIP codes, with automatic zone detection.
- Dynamic Zone Detection: Zones are automatically detected based on the ZIP codes entered, ensuring accurate results.
- Interactive Map: The map dynamically highlights the regions based on the ZIP codes entered, providing a visual guide to the shipping zones.
- Data Validation: The tool checks the validity of the ZIP codes entered to ensure accuracy.
- Instant Feedback: Receive quick feedback about your entered ZIP codes, including the associated city and state.
- Mobile Responsive: Designed with a mobile-first approach, the tool works seamlessly across all devices.

Installation & Running the App Locally (Mac and Windows):
1. Open the Terminal
2. Install http-server if not already installed
3. Navigate to the project directory
4. In the project directory, run the command 'http-server'. The terminal will output where the app is running on your local machine.
